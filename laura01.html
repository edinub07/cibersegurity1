Tipos de Malware

Regresar
Tipo de Malware Función Impacto
Virus Autoreplicable, infecta archivos Daño de datos y sistema
Gusanos Se propaga a través de redes Interrupción, agotamiento de recursos
Caballo de troya Se disfraza de software legítimo Robo de datos, control del sistema
Ransomware Cifra archivos y exige pago Pérdida de acceso a datos
Software espía Supervisa en secreto al usuario Violación de privacidad
Adware Muestra anuncios no deseados Molestia, bajo rendimiento
Rootkit Oculta software malicioso Acceso no autorizado
Keylogger Registra pulsaciones Robo de credenciales
Fileless Malware Corre en memoria Ataques difíciles de detectar
Cryptojacker Minado de criptomonedas Secuestro de recursos
Botnet Red de dispositivos infectados Ataques DDoS, spam
Bomba lógica Ejecuta código al activarse Destrucción de datos
Wiper Destruye datos Pérdida irreversible
